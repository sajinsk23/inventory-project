# Inventory Visibility Engine

## Tech Stack
- Python
- Pandas
- SQLite
- Flask
- Streamlit

## Features
- Inventory tracking
- Aging calculation
- Dead stock detection
- Dashboard KPIs and charts
- API snapshot

## How to Run
python main.py
python api.py
streamlit run dashboard.py
